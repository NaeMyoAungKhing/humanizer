# Voice Profile Template

> **This is a starting point, not a definitive description of any group, dialect, register, or culture.**
> A voice profile is a working hypothesis about a writer's signal. It exists to help a language model
> *restore* voice that automated smoothing has flattened — not to impose a stereotype.
> If you author a profile for a community you do not belong to, ground it in cited linguistic
> research, mark it as a starting point, and invite correction.

---

## 1. Identity

**Profile name:** *e.g. "American Business English — Plain-Language Tradition"*
**Intended scope:** *Who this profile is meant to fit (e.g. "first-language English writers in
US corporate and non-profit settings, professional written register, 2010–present").*
**Out of scope:** *What this profile does **not** claim to describe (e.g. "spoken register, legal
writing, fiction, regional dialects").*

## 2. Disclaimer

This profile is a starting point. It is shaped by published linguistic research (see Bibliography)
and is intended to be edited, forked, or replaced by anyone whose voice it tries to describe.
It is **not** an essentialist statement about how a group writes, and it should not be used to
"correct" anyone whose voice does not match it.

## 3. Voice signatures — PRESERVE

Concrete, observable features that the model should treat as *signal*, not noise.
List 8–15. Each one needs a one-line description and at least one example.

- **Signature name** — short description.
  - Example: *short example sentence demonstrating the feature*.

## 4. AI-smoothing patterns — REVERSE

Patterns that automated text generation or AI editing tends to introduce, that flatten this voice.
For each pattern, give the smoothed form and the restored form.

- **Pattern name** — what gets added or removed by smoothing.
  - Smoothed: *what AI tends to produce*
  - Restored: *what this voice would actually do*

## 5. Sentence rhythm

Describe the typical distribution of sentence lengths, and the *variation* across a paragraph.
Use ranges, not single numbers. Note any characteristic short/long alternation.

- Median sentence length: *e.g. 14 words*
- Range: *e.g. 4–32 words*
- Rhythm note: *e.g. "frequent one-clause sentences followed by a longer subordinated one"*

## 6. Lexical preferences

- **Prefer:** *short list of word choices, registers, or collocations characteristic of this voice*
- **Avoid:** *short list of words/phrases that AI smoothing tends to insert, that this voice would not*

## 7. Discourse markers

How this voice signals turn-taking, hedging, contrast, emphasis in *writing*. Particles,
connectives, transitional phrases, capitalisation patterns, punctuation conventions.

## 8. Punctuation and orthography

Anything load-bearing about how this voice handles capitalisation, em-dashes, ellipses,
parentheses, line breaks, etc. Do not over-prescribe — note tendencies, not rules.

## 9. What this profile is NOT trying to do

A short paragraph naming the things this profile explicitly does not attempt: e.g. "this is not
a profile for spoken Singlish", "this is not a profile for academic prose", "this is not a
profile for any one named individual".

## 10. Sample restoration

A short before/after pair. The "before" should be a paragraph in flat, AI-smoothed prose.
The "after" should be the same content rewritten in the voice this profile describes.

**Before (smoothed):**

> *paragraph in generic AI-edited prose*

**After (restored):**

> *same content, written in this voice*
